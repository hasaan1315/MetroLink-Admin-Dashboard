<h1>MetroLink Admin Dashboard – Management System</h1>

<h2>Description</h2>
The MetroLink Admin Dashboard is a web-based management panel developed to control and monitor the services provided by the MetroLink smart transportation system. It allows administrators to manage user data, respond to lost item reports, publish service alerts, monitor payments, and review user feedback in real time. Built using HTML, CSS, and JavaScript, and connected to Firebase for backend operations, it provides seamless communication between admins and system data.

<h2>Languages and Technologies Used</h2>
- <b>HTML</b> (Structure and Content)  
- <b>CSS</b> (Styling and Layout)  
- <b>JavaScript</b> (Functionality)  
- <b>Firebase</b> (Authentication, Firestore DB)  

<h2>Environments Used</h2>
- <b>Visual Studio Code</b>  
- <b>Firebase Console</b>  
- <b>Web Browsers (Chrome, Edge)</b>

<h2>Key Features</h2>

- 👤 <b>Admin Authentication:</b> Secure sign-up/login using Firebase Auth  
- 👥 <b>User Management:</b> Monitor user data and commuter profiles  
- 🧳 <b>Lost Item Handling:</b> View, verify, and update lost item cases  
- 📢 <b>Service Alerts:</b> Post real-time metro updates and maintenance alerts  
- 💳 <b>Card & Payment Management:</b> Review virtual card balances and top-up history  
- 📩 <b>Feedback System:</b> Respond to user complaints and support requests  
- 🔐 <b>Security Logs:</b> Track access and ensure dashboard integrity  
- 🔄 <b>Firestore Sync:</b> Real-time updates using Firebase Firestore

<h2>Conclusion</h2>
The MetroLink Admin Dashboard is the digital backbone of the MetroLink ecosystem, offering administrators the tools needed to manage, respond, and act with precision. It complements the mobile app and ensures full lifecycle support for commuters and staff alike.
